关于Hexo博客部署到GitHub一直没有contribution的解决办法

Dec 6, 2017 | 阅读

Hexo搭建的博客系统托管在Github上之后一直没有contribution，今天偶然发现是~/.gitconifg中的email与Github上的email不一致导致的。
改成一致之后还需要重新生成ssh秘钥，将公钥添加到远程仓库后才能正常部署。
了解详细的信息，请访问https://help.github.com/articles/why-are-my-contributions-not-showing-up-on-my-profile/
