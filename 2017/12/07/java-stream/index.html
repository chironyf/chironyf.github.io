Java的输入输出流按读写方式可以分为字节(Byte)型和字符(Character)型两种，其中字节型的顶层抽象类包括InputStream和OutputStream，字符型的顶层抽象类包括Reader和Writer。为了从内部编码的角度更好地理解二者的区别，我们先看一下这段代码。

这段代码是什么意思呢？
通过查看JDK源码，System.in是InputStream声明的对象，它是源头指向键盘的输入流，并且，是以字节形式读入的。那么，调用read()方法时，返回的值应该是”Hi 你好”对应的字节码（UTF-8）。
由于Java的编码方式是基于Unicode字符集实现的，所以先查一下字符串对应的Unicode编码
