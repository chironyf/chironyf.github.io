Java之字符串辨析

Dec 4, 2017 | 阅读

Java中的字符串有String, StringBuffer和StringBuilder三种。下面讨论一下它们各自的特点以及相互之间的区别。

String

String是被final修饰的类，不能被继承，所以我们不能扩展它的功能，并且其所创建的对象也是不能被修改的。字符串对象初始化的时候，每一个字符会被保存在一个char数组里面（value），这个数组也被final修饰，一旦被初始化之后就不能再次赋值。类中有很多对字符串的操作，例如concat(), trim()等，这些都不是对字符串对象本身的修改，而是创建了新的对象，例如JDK中的concat()函数，功能是将参数中的str追加到本字符串后面，他并没有对对象本身进行修改。

StringBuffer与StringBuilder
